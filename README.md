# Бот "Без него" (@beznegbot)

Telegram бот для отправки сообщений, которые видят **все, кроме** указанных пользователей.

## Установка и запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить бота
python bot.py
```

## Использование

### Inline режим (в любом чате)
```
@beznegbot Привет всем! @vasya @petya
```
Сообщение увидят все, кроме @vasya и @petya.

### Команды в ЛС с ботом
- `/start` — инструкция
- `/setmessage <текст>` — изменить текст для исключённых
- `/resetmessage` — сбросить на стандартный
- `/settings` — текущие настройки

## Как это работает

1. Пишешь `@beznegbot сообщение @username1 @username2` в любом чате
2. Выбираешь результат из выпадающего меню
3. В чат отправляется сообщение с кнопкой "Показать"
4. При нажатии:
   - Исключённые видят "Это сообщение не для тебя"
   - Остальные видят само сообщение

## Файлы

- `bot.py` — основной код бота
- `requirements.txt` — зависимости
- `bezneg_bot.db` — база данных SQLite (создаётся автоматически)
